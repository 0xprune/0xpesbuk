## Requirements

* PHP 8.2 or later
* Composer
* Git
* Terminal
* MySQL
* VSCode / any IDE

## Installation

1-Clone the project to your htdocs directory

2-Configure the Database (app/config/Database.php)

3-Configure the config file (app/config/App.php)

## Running project
on terminal: `php spark serve`
